- ## 功能

	- Audio
	- Video

- ## 用法
	- 输入斜杠命令: `/Save link assets to local`
		- ![command](./assets/command.jpg)
	- 点击块级菜单项: `Save link assets to local`
		- ![block_menu_item](./assets/block_menu_item.jpg)
	- 点击页面菜单项: `Save all link assets to local`
		- ![page_menu_item](./assets/page_menu_item.jpg)
